* keeps detailed history
* helps tracks changes of files ovetime
* avoid confusing file versions and change manually

### Github
* web service
* runs on Git software servers
* can let users manage their projects online
* it enhances the experience of collaboration
* social interactions around shared work to be easier and more effective
  
### Repo
* digital asset center that holds files and history
* easy for organizing and managing
* version control

### Commit
* save point of file at a specific time
* version control allowing managing, tracking, and changes throughout the project history
* commits save
